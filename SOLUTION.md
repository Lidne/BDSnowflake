# BigDataSnowflake
Анализ больших данных - лабораторная работа №1 - нормализация данных в снежинку

В работе используется исходная таблица `mock_data`, в которую загружаются данные из CSV-файлов.

На ее основе построена структура в формате звезды.

Факт:
`sales` - таблица продаж. Содержит дату продажи, количество, общую стоимость и ссылки на измерения.

Измерения:
`customers` - информация о покупателях.
`sellers` - информация о продавцах.
`products` - информация о товарах.
`stores` - информация о магазинах.
`pets` - категория питомца.
`suppliers` - информация о поставщиках.

Связь между таблицами реализована через первичные и внешние ключи. Таблица `sales` связана с таблицами измерений по полям `customer_id`, `seller_id`, `product_id`, `store_id`, `pet_id`, `supplier_id`.
